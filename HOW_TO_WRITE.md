# ✍️ IBAS 팀 블로그 글 작성 방법 (Decap CMS)

1. 링크 열기: **https://<네틀리파이-배포주소>/admin**
2. 이메일로 로그인 (초대받은 계정)
3. `New Post` 클릭 → 제목, 본문, 이미지 작성
4. `Save`(초안) → `Publish`(발행)
5. 발행 후 1~2분 내로 **https://ibas-dev-project.github.io** 에 자동 반영

**이미지 경로는 자동 관리**되며, 코드 블록은 아래처럼 작성하면 됩니다:
```ts
console.log("Hello IBAS");
```
